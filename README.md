# Welcome to SUPerman

**SUPerman** is a cutting-edge tool designed for the fast computation of matrix permanents, supporting both sparse and dense matrices. Leveraging the power of CPU, GPU, and distributed computing environments, SUPerman provides efficient and scalable solutions for exact and approximation algorithms.

*Please note that this project is still in progress.*

Supported platforms: gcc 9.4.0, CUDA 10.1
